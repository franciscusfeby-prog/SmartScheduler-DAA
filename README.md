# SmartScheduler-DAA

## Overview
SmartScheduler adalah proyek mini dari mata kuliah Desain dan Analisis Algoritma 2025.
Program ini menjadwalkan job agar total waktu penyelesaian minimum dan sebanyak mungkin job selesai sebelum deadline.

## Algoritma
Menggunakan pendekatan Greedy – Earliest Deadline First (EDF).
Kompleksitas waktu: O(n log n).

## Cara Menjalankan
1. Kompilasi file main.cpp dengan perintah:
   ```bash
   g++ main.cpp -o SmartScheduler
   ```
2. Jalankan program:
   ```bash
   ./SmartScheduler
   ```
3. Masukkan jumlah job, processing time, dan deadline sesuai instruksi.
